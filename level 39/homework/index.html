Color Systems: RGB, HEX, HSL

RGB Color System

RGB (Red, Green, Blue) არის ფერების მოდელი, რომელიც იყენებს წითელ, მწვანე და ლურჯ კომპონენტებს, რათა შექმნას სხვადასხვა ფერები. თითოეული კომპონენტის მნიშვნელობაა 0-დან 255-მდე, რაც განისაზღვრება შემდეგნაირად:
rgb(255, 0, 0) ქმნის წითელ ფერს.

HEX Color System

HEX ფერის კოდები წარმოდგენილია ექვსციფრიანი კოდით, რომელიც იწყება # ნიშნით.
თითოეული ორი ციფრი წარმოადგენს ფერს (წითელი, მწვანე და ლურჯი) 16-ბიტიან ფორმატში (00-დან FF-მდე).
მაგალითად: #FF0000 ქმნის წითელ ფერს.

HSL Color System

HSL (Hue, Saturation, Lightness) არის ფერების მოდელი, რომელიც იყენებს ელფერს, სინთეზსა და სინათლეს ფერის შესაქმნელად.
მაგალითად: hsl(0, 100%, 50%) ქმნის წითელ ფერს.
